# 21天是什么?

21天是一个结合课程表的习惯养成APP

# 21天的功能

“21天”是通过分析用户使用手机的行为，根据用户的日常作息，合理安排和利用时间，在校学生可以结合自己的课程表的时间分配，就职员工可以结合自己的工作时间，鼓励学生减少在课堂使用手机的频率督促用户养成良好习惯，提高用户工作、学习效率。
